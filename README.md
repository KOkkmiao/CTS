    根据选中的内容快速查询properties的value

## 使用教程
***只支持properties 数据***
 ```json
  {
  "headers":{
    "Accept":"*/*",
    "Accept-Language":"zh-CN,zh;q=0.9",
    "Cache-Control":"no-cache",
    "Content-Type":"application/json",
    "Pragma":"no-cache",
    "X-Requested-With":"XMLHttpRequest",
    "Cookie":"" #用户 cookie 必填
  },
  "body":null,
  "method":"GET",
  "urls":{ # 多个请求配置的数据
    "001-application":"",
    "002-rocketmq":""
  },
  "parse-body":"xxx.xxx.xxxx" # 全量数据节点
}
 ```
